clientcord

extremely fast. very low ram. fully typed. totally modular. decorator-driven discord framework for python 3.12+.

what is clientcord

clientcord is a from-scratch discord bot framework that prioritizes performance and developer experience. it uses decorator-driven architecture inspired by modern frameworks like fastapi and seyfert. it does not imitate discord.py.

core libraries

  • msgspec - fast json parsing and typed structs
  • aiosonic - async http requests
  • websockets - gateway connection
  • isal - zlib decompression
  • uvloop - fast event loop (linux only)
  • lmdb - disk-backed cache for low ram
  • xxhash - fast hashing
  • psutil - memory tracking

installation

clientcord is lightweight and easy to install. just run this in your terminal:

terminal
pip install msgspec aiosonic websockets isal uvloop aiodns lmdb xxhash colorama psutil
