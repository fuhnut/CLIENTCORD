modals

modals are popup forms that request structured input from users. they can contain text inputs, select menus, and static text. all interactive components inside a modal must be wrapped in a Label.

creating a modal

basic modal structure
from clientcord.components import Modal

modal = (
    Modal()
    .set_title("my modal")
    .set_custom_id("my_first_modal")
    .set_components([])
)

each interactive component must be wrapped in a Label:

wrapping with a label
from clientcord.components import Label

label = (
    Label()
    .set_label("some field")
    .set_component(my_component)
)

1. text input modal

the classic discord modal with a text field.

text input modal
from clientcord.components import Modal, Label, TextInput, TextInputStyle

text_input = (
    TextInput()
    .set_custom_id("feedback")
    .set_style(TextInputStyle.paragraph)
    .set_placeholder("write your feedback here...")
    .set_length(min=10, max=500)
    .set_required(True)
)

label = (
    Label()
    .set_label("what did you think?")
    .set_description("give us as much detail as possible")
    .set_component(text_input)
)

modal = (
    Modal()
    .set_title("feedback form")
    .set_custom_id("feedback_modal")
    .set_components([label])
)

sending the modal

sending from a command
await ctx.show_modal(modal)

2. static text in modal

use TextDisplay to show read-only text inside a modal.

text display in modal
from clientcord.components import Modal, TextDisplay

text = TextDisplay().set_id(1).set_content("read these rules before continuing")

modal = (
    Modal()
    .set_title("rules")
    .set_custom_id("rules_modal")
    .set_components([text])
)

3. string select menu modal

string select in modal
from clientcord.components import (
    Modal, Label, StringSelectMenu, StringSelectOption
)

menu = (
    StringSelectMenu()
    .set_custom_id("color")
    .set_options([
        StringSelectOption().set_label("red").set_value("red"),
        StringSelectOption().set_label("blue").set_value("blue"),
    ])
)

label = Label().set_label("pick a color").set_component(menu)

modal = (
    Modal()
    .set_title("color picker")
    .set_custom_id("color_modal")
    .set_components([label])
)

2. Handing the Modal Result

just like buttons, you create a handler in /components to process the form when it's submitted.

components/feedback_handler.py
from clientcord.components import ModalCommand
from clientcord.declare import modal

@modal(custom_id="feedback_modal")
class FeedbackModal(ModalCommand):
    async def run(self, ctx):
        # Drill into the payload to get the user's input
        data = ctx.interaction.get("data", {}).get("components", [])
        # clientcord keeps the structure simple to keep RAM low!
        await ctx.write("thanks for your feedback!")
