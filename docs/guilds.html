guilds

guilds in discord are essentially the servers that contain users, channels, roles, and emojis. clientcord maps guild information using the Guild model.

fetching a guild

via context
# standard method if available in the context payload
guild = ctx.guild

moderating users

the guild model exposes high-level moderation actions to seamlessly remove or penalize users.

banning a member
await ctx.guild.ban(
    "user_id",
    delete_message_seconds=86400, # Delete last 24h of messages
    reason="spamming the chat"
)
unbanning a user
await ctx.guild.unban("user_id", reason="appealed")
kicking a member
await ctx.guild.kick("user_id", reason="inactive")

guild properties

your Guild model will contain critical properties about the server.

  • id - the guild id
  • name - the guild name
  • owner_id - id of the server owner
  • member_count - total members (approximate if full data is missing)
  • icon - the guild icon hash
  • features - list of guild features (e.g., 'ANIMATED_ICON')
