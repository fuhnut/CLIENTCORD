command options

options let users pass information to your commands (like a name, a number, or a file). clientcord makes this super easy with two ways to do it.

The Easy Way: Dictionary Syntax

this is the fastest way to add options. you just pass a simple dictionary with the type and description. clientcord handles all the boring stuff for you!

commands/kick.py
from clientcord.declare import command, options

@options(
    target={"type": "user", "description": "the user to kick", "required": True},
    reason={"type": "text", "description": "why you are kicking them"}
)
@command(name="kick", description="kick a member")
class KickCommand:
    async def run(self, ctx, target, reason="no reason provided"):
        # 'target' is automatically converted to a User object!
        await ctx.write(f"kicked {target.username} for {reason}")

supported type names

  • "text" or "string"
  • "integer" (whole numbers)
  • "bool" (true/false)
  • "user" (server members)
  • "channel" (text/voice channels)
  • "role" (guild roles)
  • "attachment" (files/images)

The Advanced Way: Option Constructors

if you need more control, you can use built-in functions for each type. this is great for adding choices or specific requirements.

string options

commands/greet.py
from clientcord.declare import command, options
from clientcord.commands.options import string_option

@options(
    name=string_option(description="who to greet", required=True)
)
@command(name="greet", description="greet someone")
class GreetCommand:
    async def run(self, ctx, name: str):
        await ctx.write(f"hello {name}!")

integer options

commands/roll.py
from clientcord.declare import command, options
from clientcord.commands.options import integer_option

@options(
    sides=integer_option(description="number of sides", required=False)
)
@command(name="roll", description="roll a die")
class RollCommand:
    async def run(self, ctx, sides: int = 6):
        import random
        result = random.randint(1, sides)
        await ctx.write(f"rolled a {result}")

boolean options

commands/ping.py
from clientcord.declare import command, options
from clientcord.commands.options import boolean_option

@options(
    hide=boolean_option(description="hide the response")
)
@command(name="ping", description="show latency")
class PingCommand:
    async def run(self, ctx, hide: bool = False):
        await ctx.write("pong")

user options

commands/info.py
from clientcord.declare import command, options
from clientcord.commands.options import user_option
from clientcord.models import User

@options(
    target=user_option(description="user to look up", required=True)
)
@command(name="info", description="get user info")
class InfoCommand:
    async def run(self, ctx, target: User):
        await ctx.write(f"user id: {target.id}")
