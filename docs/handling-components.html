handling components

after sending a component in a message, you need to handle the interaction when a user clicks a button or selects an option.

step 1: update your config

make sure your cc_init.py has the components folder set so clientcord knows where to auto-load handlers from.

cc_init.py
from clientcord.intents import Intents

token = "your-token"
intents = Intents.default()
prefix = "!"

commands = "/commands"
components = "/components"
events = "/events"

step 2: create a component handler

create a file inside the /components directory. make a class that extends ComponentCommand and set the component_type to the type you want to handle.

components/hello_button.py
from clientcord.components import ComponentCommand

class HelloWorldButton(ComponentCommand):
    component_type = "Button"

step 3: filter component interactions

you want the handler to only respond to the specific component. use the filter method and check the custom_id.

components/hello_button.py
from clientcord.components import ComponentCommand

class HelloWorldButton(ComponentCommand):
    component_type = "Button"

    def filter(self, ctx):
        return ctx.custom_id == "hello-world"

step 4: run the handler

if filter returns True, clientcord executes the run method.

components/hello_button.py
from clientcord.components import ComponentCommand

class HelloWorldButton(ComponentCommand):
    component_type = "Button"

    def filter(self, ctx):
        return ctx.custom_id == "hello-world"

    async def run(self, ctx):
        await ctx.write({"content": "hello world"})
component handlers are auto-loaded from the /components folder. no manual registration needed. each file should contain one handler class.

handling select menus

works the same way. just set component_type to the select type.

components/color_picker.py
from clientcord.components import ComponentCommand

class ColorPicker(ComponentCommand):
    component_type = "StringSelect"

    def filter(self, ctx):
        return ctx.custom_id == "color_pick"

    async def run(self, ctx):
        selected = ctx.values[0]
        await ctx.write({"content": f"you picked {selected}"})

available component types for handlers

  • "Button"
  • "StringSelect"
  • "UserSelect"
  • "RoleSelect"
  • "ChannelSelect"
  • "MentionableSelect"
