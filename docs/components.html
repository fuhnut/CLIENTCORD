building components

components are built with builders. each builder is a python class that represents a component. you customize it by chaining .set_ methods.

every interactive component must have a custom_id - a unique identifier used for handling interactions when a user clicks or selects something.

action row

all builders are sent inside an ActionRow. each message can contain a maximum of 5 action rows.

creating an action row
from clientcord.components import ActionRow

row = ActionRow().set_components([]).add_components()

buttons

an action row should contain no more than 5 buttons.

building a button
from clientcord.components import ActionRow, Button, ButtonStyle

button = (
    Button()
    .set_custom_id("first-button")
    .set_style(ButtonStyle.primary)
    .set_label("first button")
)

row = ActionRow().set_components([button])

sending components

after creating the components, send the action row in the components field.

commands/ping.py
from clientcord.declare import command
from clientcord.components import ActionRow, Button, ButtonStyle

@command(name="ping", description="show the ping")
class PingCommand:
    async def run(self, ctx):
        row = ActionRow().set_components([
            Button()
            .set_custom_id("pingbtn")
            .set_label("pong")
            .set_style(ButtonStyle.primary)
        ])

        await ctx.write({
            "content": "pong",
            "components": [row.to_dict()]
        })

string select menu

building a string select
from clientcord.components import (
    ActionRow, StringSelectMenu, StringSelectOption
)

menu = (
    StringSelectMenu()
    .set_custom_id("color_pick")
    .set_placeholder("pick a color")
    .set_options([
        StringSelectOption().set_label("red").set_value("red"),
        StringSelectOption().set_label("blue").set_value("blue"),
        StringSelectOption().set_label("green").set_value("green"),
    ])
)

row = ActionRow().set_components([menu])

text display (components v2)

text display renders markdown text. requires the IS_COMPONENTS_V2 flag.

using text display
from clientcord.components import TextDisplay, build_components_payload

payload = build_components_payload([
    TextDisplay().set_content("# hello there"),
    TextDisplay().set_content("this uses components v2"),
])

sections with thumbnails

sections associate text content with an accessory like a thumbnail image.

building a section
from clientcord.components import Section, TextDisplay, Thumbnail, build_components_payload

payload = build_components_payload([
    Section()
    .set_components([
        TextDisplay().set_content("# user profile"),
        TextDisplay().set_content("username: bobby"),
        TextDisplay().set_content("level: 42"),
    ])
    .set_accessory(
        Thumbnail().set_url("https://example.com/avatar.png")
    )
])

containers

containers visually group components with an optional accent color.

building a container
from clientcord.components import (
    Container, TextDisplay, Separator, ActionRow, Button, ButtonStyle,
    build_components_payload
)

payload = build_components_payload([
    Container()
    .set_accent_color(0x5865F2)
    .set_components([
        TextDisplay().set_content("# server rules"),
        TextDisplay().set_content("1. be respectful"),
        TextDisplay().set_content("2. no spam"),
        Separator(),
        ActionRow().set_components([
            Button()
            .set_custom_id("agree")
            .set_label("i agree")
            .set_style(ButtonStyle.success)
        ])
    ])
])

media gallery

building a media gallery
from clientcord.components import MediaGallery, MediaGalleryItem, build_components_payload

payload = build_components_payload([
    MediaGallery().set_items([
        MediaGalleryItem().set_url("https://example.com/img1.png").set_description("first"),
        MediaGalleryItem().set_url("https://example.com/img2.png").set_description("second"),
    ])
])

all component types

  • TextDisplay - markdown text content
  • Button - clickable button (primary, secondary, success, danger, link)
  • ActionRow - row container for buttons and selects
  • StringSelectMenu - dropdown with custom text options
  • UserSelectMenu - dropdown auto-populated with server users
  • RoleSelectMenu - dropdown auto-populated with server roles
  • MentionableSelectMenu - dropdown for users and roles
  • ChannelSelectMenu - dropdown auto-populated with channels
  • TextInput - free-form text (modals only)
  • Section - text with an accessory (thumbnail or button)
  • Container - visual grouping with accent color
  • Thumbnail - small image accessory
  • MediaGallery - image gallery
  • FileComponent - attached file display
  • Separator - vertical spacing divider
  • Label - wraps a component with a label (modals)
  • Modal - popup form with labeled components
