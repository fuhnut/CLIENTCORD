creating your first command

the main input for any discord bot is commands. clientcord will handle loading and executing all commands you create inside the /commands folder.

step 1: create a command file

all commands must use the @command decorator and a class with a run method.

commands/ping.py
from clientcord.declare import command

@command(name="ping", description="show bot latency")
class PingCommand:
    async def run(self, ctx):
        await ctx.write("pong")
since your cc_init.py has commands = "/commands", this file will be auto-loaded when the bot starts. you do not need to manually register it.

step 2: run your bot

terminal
python bot.py

when the bot starts, clientcord compiles execution plans for all commands and connects to discord. you should see the /ping command available on your server.

using options

commands can accept user inputs through options. use the @options decorator and option constructors.

commands/ping.py
from clientcord.declare import command, options
from clientcord.commands.options import boolean_option

@options(
    hide=boolean_option(description="hide the response")
)
@command(name="ping", description="show bot latency")
class PingCommand:
    async def run(self, ctx, hide: bool = False):
        await ctx.write("pong")
