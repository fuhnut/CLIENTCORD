complete bot example

this guide brings everything together. we will create a full, working bot with a command, a ready event, and a message logger.

1. project structure

create a folder for your bot and set up these files and directories exactly like this:

commands/
  ping.py
components/
  ping_handler.py
events/
  ready.py
  message.py
  member_join.py
cc_init.py
bot.py

2. configuring your bot (cc_init.py)

this file defines how your bot behaves. it tells clientcord where to find your commands, events, and component handlers.

cc_init.py
from clientcord.intents import Intents

token = "YOUR_BOT_TOKEN_HERE" # Get this from discord.com/developers
intents = Intents.all() # Use all intents for full functionality

# Tell clientcord which folders to watch
commands = "/commands"
events = "/events"
components = "/components"

3. the main file (bot.py)

this is the file you run to start your bot.

bot.py
from clientcord.client import Client

# Initialize the client!
client = Client()

if __name__ == "__main__":
    client.run()

4. creating a command with components (commands/ping.py)

let's create a command that responds with a button!

commands/ping.py
from clientcord.declare import command
from clientcord.components import ActionRow, Button, ButtonStyle

@command(name="ping", description="ping with a button!")
class PingCommand:
    async def run(self, ctx):
        # Create a button!
        row = ActionRow().set_components([
            Button()
            .set_custom_id("ping_btn")
            .set_label("click me!")
            .set_style(ButtonStyle.primary)
        ])

        await ctx.write("pong!", components=[row])

5. handling component interactions (components/ping_handler.py)

now we need to tell the bot what to do when that button is clicked.

components/ping_handler.py
from clientcord.declare import component
from clientcord.components import ComponentCommand

@component(custom_id="ping_btn")
class PingHandler(ComponentCommand):
    async def run(self, ctx):
        # This runs when the button with ID 'ping_btn' is clicked!
        await ctx.write("you clicked the button!", ephemeral=True)

6. setting up events (events/ready.py, message.py & member_join.py)

these files handle global events like the bot starting, messages being sent, or new members joining.

events/ready.py
from clientcord.declare import event

@event(name="READY")
async def on_ready(client, payload):
    print(f"Logged in as {client.user.username}!")
    await client.sync_commands()
events/message.py
from clientcord.declare import event

@event(name="MESSAGE_CREATE")
async def on_message(client, message):
    if message.author.bot: return
    print(f"New message from {message.author.username}: {message.content}")
events/member_join.py
from clientcord.declare import event

@event(name="GUILD_MEMBER_ADD")
async def on_member_join(client, payload):
    # 'payload' contains user and member data
    username = payload.get("user", {}).get("username")
    print(f"Welcome to the server, {username}!")

7. running your bot

open your terminal in your project folder and run:

terminal
python bot.py

your bot is now alive! it handles slash commands, interactive buttons, message logging, and welcomes new members automatically.
