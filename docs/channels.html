channels

channels represent communication spaces in discord. clientcord provides models and methods for every channel type, including text, voice, threads, forums, and media channels.

channel types

every channel has a type field. use the ChannelType class to reference them.

channel types
from clientcord.models import ChannelType

ChannelType.guild_text       # 0 - text channel
ChannelType.dm               # 1 - direct message
ChannelType.guild_voice      # 2 - voice channel
ChannelType.group_dm         # 3 - group dm
ChannelType.guild_category   # 4 - category
ChannelType.guild_announcement  # 5 - announcement
ChannelType.public_thread    # 11 - public thread
ChannelType.private_thread   # 12 - private thread
ChannelType.guild_stage_voice  # 13 - stage channel
ChannelType.guild_forum      # 15 - forum channel
ChannelType.guild_media      # 16 - media channel
example
# typically you get channels from the ctx, e.g. ctx.channel
channel = await ctx.client.get_channel("123456789")

modifying a channel

update a channel's settings. all parameters are optional. supports the X-Audit-Log-Reason header via the reason parameter.

example
await ctx.channel.edit(
    name="new-name",
    topic="updated topic",
    reason="reorganizing channels"
)

deleting a channel

deleting a guild channel cannot be undone. for dms, closing is reversible by reopening.
example
await ctx.channel.delete(reason="cleanup")

permission overwrites

permission overwrites control per-channel access for roles and members.

example
from clientcord.models import Overwrite

await ctx.channel.edit_permissions(
    "987654321",
    allow="1024",
    deny="0",
    type="role",
    reason="allow role to view"
)
example
await ctx.channel.delete_permission("987654321")

threads

threads are temporary sub-channels. use these methods to create, join, and manage threads.

example
thread = await ctx.channel.start_thread_from_message(
    "message_id",
    name="discussion",
    auto_archive_duration=1440
)
example
thread = await ctx.channel.start_thread(
    name="general-chat",
    type="public_thread"
)

await thread_channel.remove_thread_member("user_id")
example
members = await thread_channel.list_thread_members(with_member=True)
