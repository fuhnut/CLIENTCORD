# clientcord

extremely fast. very low ram. fully typed. totally modular.

## installation

```bash
pip install clientcord
```

## features

- decorator driven architecture
- completely modular importing (no unneeded caching)
- highly efficient disk-backed storage using lmdb
- full intent/permission bitflag safety

see `docs/index.html` for a full step by step walkthrough on creating your clientcord bot.
